 # Compose Key (CK)

[Ubuntu Basics SWAP KEYS 3 compose key & 3rd level key - Tutoringby Pop](https://www.youtube.com/watch?v=5Q_4rod5GWQ)  
[ComposeKey](https://help.ubuntu.com/community/ComposeKey)  
[Compose key - Wiki](https://en.wikipedia.org/wiki/Compose_key)
[Where can I find the full list of Compose combinations for my locale?](https://askubuntu.com/questions/34932/where-can-i-find-the-full-list-of-compose-combinations-for-my-locale)  

ä: compose, then " then a
ö: compose, then " then o
ü: compose, then " then u
Ä/Ö/Ü: compose, ", then Shift+a/o/u
ß: compose, then s then s (or another sequence depending on locale; some setups use compose, then " then s
£: shift+3
#: space+# 

> Examples

- ü : press down the CK [AltGR] first, then press the key u, this prints u, then press [Shift+"] which turns the u into ü.
- ä : same as above.
- ö : same as above.

- Ä : this works the same but you must first press the [Shift] key to print the uppercase char A, then press [Shift+"] which turns the A into Ä.
